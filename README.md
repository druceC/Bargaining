# Fund Vanishes 💰 

This repo contains the code for an **economics experiment** on decision-making and group dynamics for Professor Andrzej Baranski.

## Experiment Overview

This experiment explores how people make decisions about splitting shared resources.

- **Goal**: Understand group decision-making and resource division.
- **Participants**: You're placed in a randomly assigned **group of 3**.
- **Task**: As a group, decide how to split **30 tokens**.
  - **Exchange Rate**: `4 tokens = 1 USD`

### Two Main Stages

Each round has two parts: **Proposal** and **Vote**.

#### 1. Proposal
- Everyone submits a proposed split of the 30 tokens.
- Shares must:
  - Add up to 30 tokens.
  - Be **whole numbers** (integers only).
- One proposal is picked **at random** to move to the vote.

#### 2. Vote
- Everyone sees the selected proposal and votes to **accept** or **reject** it.
- The proposal passes if **at least 2 out of 3** vote to accept.
- The proposer’s vote **automatically counts as a ‘yes’**.

### Outcomes

- **If Approved**: Tokens are divided based on the proposal.
- **If Rejected**: Tokens **vanish** — everyone gets `0`.

### Rounds & Payment

- The experiment runs for **3 rounds**.
- You are **randomly reassigned** to a new group each round.
- Only **1 round is randomly selected** for actual payment.
